Chào mừng đến trang quản trị DemoTiến. Bạn có thể:

  • Danh sách sản phẩm — Xem, tạo, sửa, xóa sản phẩm
  • Quản lý liên hệ — Xem danh sách người liên hệ, đánh dấu đã xử lý
  • Thêm nội dung — Sử dụng form ở trang sản phẩm để thêm sản phẩm mới

Thêm nội dung sau này kiểu gì?

📝 Cách 1: Qua giao diện web (Editor)

  • Vào /admin/products → Nút "Tạo sản phẩm mới"
  • Điền form rồi save → Sản phẩm xuất hiện tức thì

🗄️ Cách 2: Trực tiếp vào Database

  • Kết nối PostgreSQL qua DBeaver / pgAdmin
  • Chọn bảng product, insert dữ liệu trực tiếp
  • Reload trang web để thấy cập nhật

🔗 Cách 3: Qua API từ bên ngoài

  • POST /api/admin/products với JSON body
  • Tiện cho automation / mobile app gọi
