Hướng dẫn nhanh

Chào mừng đến trang quản trị DemoTiến. Bạn có thể:

  • Danh sách sản phẩm — Xem, tạo, sửa, xóa sản phẩm
  • Quản lý liên hệ — Xem danh sách người liên hệ, đánh dấu đã xử lý
  • Thêm nội dung — Sử dụng form ở trang sản phẩm để thêm sản phẩm mới

Cách Thêm Nội Dung

5 cách dễ nhất:

  1. Web UI — Vào /admin/products.html → "Tạo sản phẩm mới" → Điền form → Save
  2. Database — Kết nối PostgreSQL, INSERT vào bảng product
  3. API — POST /api/admin/products với JSON body
  4. Sidebar — Chỉnh sửa file HTML products.html phần sidebar (bên trái)
  5. Trang mới — Tạo controller + template + CSS mới
